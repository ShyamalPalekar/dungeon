Dungeon Game - Assets
--------------------------------
Place all files in your WPF project's Assets/ folder.

Files:
- bgm.wav       : Ambient 12s loop (seamless).
- click.wav     : UI click.
- hover.wav     : UI hover.
- move.wav      : Board move.
- win.wav       : Victory jingle.
- lose.wav      : Defeat sting.
- logo.png      : Title/home logo.
- splash.png    : Welcome/tutorial backdrop.
- tile_orb.png  : Extra tile icon.

All assets are procedurally generated for this project and licensed to you for use in this game.
